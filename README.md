## Medium Clap Demo

![GIF of Medium clap demo](assets/explainer-demo.gif)

## Run project

1. Change directory

```sh
cd showcase
```

2. Install dependencies

```sh
npm install
or
yarn
```

4. Run the app

```sh
npm dev
or
yarn dev
```
